Understanding Transaction Types

Transaction types like {{ transaction_type.transaction_code }} are used in AML rules to target specific kinds of financial activity.

When a rule is configured with a transaction type or group that includes {{ transaction_type.transaction_code }}, the monitoring system will evaluate transactions with this code against the rule's criteria.

The transaction type grouping system allows rules to monitor entire categories of transactions (e.g., all cash deposits, all international wires) without needing to specify each individual transaction code.
